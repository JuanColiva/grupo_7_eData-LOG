# grupo_7_eData-LOG
# Objetivo del sitio: 
eData-LOG permite capturar los datos públicos de los clientes que acceden al Wifi de tu local para generar una base de datos segmentada y hacer acciones de marketing productivas. Nuestro servicio va dirigido a bares, restaurantes, shoppings y cualquier negocio que esté en permanente interacción con sus clientes.
